# 📊 Business Sales Prediction Using Machine Learning

A Flask-based machine learning application that predicts product sales using a Random Forest Regression model.

## Features

- Sales prediction
- Exploratory Data Analysis (EDA)
- Random Forest Regression
- Interactive Flask web application
- Business analytics dashboard

## Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- HTML
- CSS

## Project Structure

```
Sales_Prediction/
├── dataset/
├── graphs/
├── model/
├── static/
├── templates/
├── app.py
├── train_model.py
├── eda.py
└── generate_dataset.py
```

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/Sales_Prediction.git

cd Sales_Prediction

pip install -r requirements.txt

python app.py
```

Open:

http://127.0.0.1:5000
